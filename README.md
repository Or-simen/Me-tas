# User Guide:
Installation Requirements & Environment:

The application was developed using android studio koala 2024.1.1 and built in Java, with Firebase used for user authentication and data storage.

The application was tested and verified to work successfully on:

Pixel 5 API 26 emulator

Medium Phone API 36 emulator

No special permissions are required to install the application. However, an active internet connection is necessary for Firebase authentication.

Application Overview

This application is designed to assist users in reporting, tracking, and receiving alerts regarding suspicious or unusual bakai (flying ATV) activity in their area.

The app includes a login screen, registration screen, and multiple user interfaces for pilots or other users. Core functionalities include:

Reporting bakai takeoff or landing

Viewing previous reports

Displaying a list of currently active bakais

Viewing unidentified alerts and reports

Sending and receiving information between users

User Notifications:

The application will display a notification when a suspicious flight is reported.
